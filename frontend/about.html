About

This project was built to provide an easy way to view and compare Fortnite stats in real time. Whether you're a casual player looking to track your progress or a competitive gamer analyzing pro stats, this tracker has got you covered. The project is powered by Flask (Python) for the backend, fetching real-time stats from the Fortnite API. The frontend is built with HTML, CSS, and JavaScript to display the data in a user-friendly way.
